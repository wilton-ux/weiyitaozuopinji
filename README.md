# Portfolio Website

A personal portfolio website with magazine-style layout, custom cursor, smooth scroll animations, and rich hover effects.

**Black background · #FFFFE3 accent · Playfair Display + Inter fonts**

---

## Quick Start

Double-click `index.html` to open the site in your browser.

---

## Deploy

### Netlify Drop (Easiest)
Drag the entire `portfolio/` folder onto [app.netlify.com/drop](https://app.netlify.com/drop).

### GitHub Pages (Free)
1. Push this folder to a repo named `yourusername.github.io`
2. Enable Pages in Settings → Pages → Source: main

### Vercel
Drag the folder onto [vercel.com/new](https://vercel.com/new).

---

## Tech Stack

- HTML + CSS + Vanilla JS (zero build tools)
- GSAP + ScrollTrigger for scroll animations
- Lenis for smooth scrolling
- Google Fonts (Noto Serif SC + Noto Sans SC)

## File Structure

```
portfolio/
├── index.html      # All content, edit your text here
├── style.css        # All styling
├── script.js        # Animations, cursor, config
├── assets/
```
